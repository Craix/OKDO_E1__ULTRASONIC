OKDO E1 ULTRASONIC

Module info: <br>
Ultrasonic distance sensor HC-SR04 <br>
LCD 128 x 160 SPI ST7735S <br>

Connection schematic: <br>

[Ultrasonic]
* Trigger - PIO0_0  <br>
* Echo -  PIO1_31 (INT) <br>
* VCC - 5V <br>

[TFT (FLXC_8  - SPI)]
* LED - VDD (3.3V)
* SCK - PIO1_2 (SCK)
* SDA - PIO0_26 (MOSI)
* A0 -  PIO1_26
* RESET - PIO1_27
* CS - PIO1_1 (SSEL 1)
* VDD - 3.3V

Common <br> 
* GND - GND
